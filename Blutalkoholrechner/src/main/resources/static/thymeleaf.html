Thymeleaf

Natural Templating

  • Normale, valide HTML Dateien
  • Können direkt im Browser angezeigt werden
  • Einfache Entwicklungs-Iterationen mit Designer:innen!
  • Vergleich Thymeleaf vs. Mustache

Attribute

  • Entweder th:attr oder data-th-attr
  • Überschreiben von Standard-Attributen
  • Attribute und Thymeleaf-Attribute gleichzeitig sinnvoll (siehe <link> Tag in dieser Datei)
  • Zusätzliche Attribute für Schleifen, Conditionals, Content-Ersetzung

Expression Language

  • Einsetzen von Ausdrücken mit ${expr}
    • Variablen: ${name}
    • Attribute von Objekten: ${user.name} (User hat ein öffentliches Attribut oder einen Getter)
    • Ergebnisse von Methodenaufrufen: ${user.berechneSaldo()}
    • Ergebnisse von Java Ausdrücken: ${user.berechneSaldo() > 10000 ? 'Goldkunde' : 'Joe Shmoe' }
    • Statische Methoden ${T(java.lang.Long).bitCount(2374682374L)}}
  • Ersetzungen in Textliteralen (Literal substitutions) |Welcome to our application, ${user.name}!|
    Es ist eingeschränkt, was erlaubt ist. Ggf. lieber mehrere <span> nutzen.
  • Besondere Hilfs-Objekte (siehe Dokumentation)
    Beispiel #calendars: ${#calendars.format(termin,'dd.MM.yyyy')} Besser thymeleaf-extras-java8time als Dependency laden.
  • URLs
    • Pfad-Parameter: @{/greeter/greet/{name}/{staerke}(name=${name}, staerke=3)}
    • Query-Parameter@{/greeter/greet(name=${name}, strength=${strength.length() * 2})}

Kontrollstrukturen

  • Bedingtes Einfügen <p th:if="${expr}">...</p>
  • Schleifen <li th:each="e : ${listExpr}" th:text="${e}">...</li>
    1. th:remove="all-but-first"
    2. th:object und *{attribute}
